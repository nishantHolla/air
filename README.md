# air

This repository contains implementations of core information retrieval algorithms developed as part
of my 6th semester coursework.

## Usage

- Clone the repository
```bash
git clone https://github.com/nishantHolla/air.git
cd air
```

- Create python virtual env and install uv
```bash
python -m venv .venv
pip install uv
```

- Install necessary packages
```bash
uv sync
```
